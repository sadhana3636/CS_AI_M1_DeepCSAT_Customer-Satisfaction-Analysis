# CS_AI_M1_DeepCSAT_Customer-Satisfaction-Analysis

# CSAT Prediction using Deep Learning (ANN)

## 📌 Overview
This project predicts Customer Satisfaction (CSAT) scores using a Deep Learning model (Artificial Neural Network). It analyzes customer interaction data to help businesses improve service quality and make data-driven decisions.

---

## 🎯 Problem Statement
Traditional CSAT surveys are slow and often incomplete. This project aims to predict customer satisfaction in real time using historical interaction data.

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- TensorFlow / Keras  

---

## ⚙️ Workflow
- Data Cleaning & Preprocessing  
- Feature Engineering  
- Model Building (ANN)  
- Model Evaluation  

---

## 📊 Key Insights
- Faster response time improves customer satisfaction  
- Longer handling time reduces CSAT  
- Service quality varies across categories and agents  

---

## 📈 Results
- The model successfully predicts CSAT levels  
- Helps identify dissatisfied customers early  
- Supports proactive decision-making  

---

## 🚀 Deployment
The model can be deployed locally to generate real-time CSAT predictions using new customer data.

---

## 📌 Conclusion
This project demonstrates how deep learning can be applied to improve customer experience and business performance in e-commerce.
